# Visa signal is a tag from public data, not a prediction model

Visa-friendliness is surfaced as a factual tag ("Company X sponsored N H1Bs last year") sourced from public USCIS H1B disclosure data — not as a probability score from a trained model. A prediction model was rejected because training data for interview-to-offer rates by nationality/visa status is not publicly available; any model built on proxies would be unreliable and potentially discriminatory. Public USCIS data is factual, auditable, and updated annually.
