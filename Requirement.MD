This application will help them to prep their job by providing them with the following features:
1. CV Reading, ATS, giving the score and what should they improve as they are applying for big company like Google, Apple, Meta, Amazon, Microsoft. I think for this, it should be a features contain multiple things: first, compared to the JD. Second, compare to the real development scence of the market, and we should have some source-of-truth so it can validate whether we are there yet. And this source-of-truth should be updated every day, week because daily or weekly, there are some new features, skills and cool projects or papers that people can implement. I think it should be a features that can read the user CV. And suggest what skills they should have and what project they should build based on a formula of 3 projects: 1 projects of building a full-stack app (paired with system design concept) with all up-to-date skills from the source of truth, an app related to deep-dive CS fundamental, and another app that require them to re-implement some scientific papers and make some practical improvements to it, for example, Maglev 2014 with update from 2016 and 2020.

2. Learning

For every applicants, they should learn a lot of things, at a lot of levels. For example, for a SWE roles. They should learn DSA, System Design, LLD, CS Fundamentals for the interview. The list may go on depending on the role: Networking, Machine Learning, ... The purpose this function is not also help them prep for the interview, but it also help them to learn what they need to know to get the real shit done. I mean, the market is always changing, so we need to have a system that can provide them with the most up-to-date knowledge they need to know. Also, the learning should be personalized to each user. For example, if a user is weak in DSA, the system should provide them with more DSA learning materials. If a user is strong in DSA, the system should provide them with more System Design learning materials. And based on the user's CV, the system should also provide them with the most relevant learning materials for their target roles.

For example, the list of learning materials should be updated every day, week because daily or weekly, there are some new features, skills and cool projects or papers that people can implement.
